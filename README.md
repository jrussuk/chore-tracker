### Cat Chores (Rotate Daily)

**Litter Tray:**
- Odd days (1st, 3rd, 5th, etc.) ‚Üí Ella
- Even days (2nd, 4th, 6th, etc.) ‚Üí Hamish

**Feed Cats:**
- Odd days ‚Üí Hamish
- Even days ‚Üí Ella

*Rule: Whoever does litter tray, the other person feeds*

### Household Chores

**Empty Dishwasher:**
- Shows up for both kids
- Whoever gets to it first logs it
- Can be done multiple times per day

**Recycling & Bins:**
- Shows up for both kids
- Weekly task
- Either child can complete

## ‚ûï How to Add New Chores

1. **Open the Google Sheet** (use link from Parent View)
2. **Go to the "Chores" tab**
3. **Add a new row** with:
   - **Next ID number** (5, 6, 7...)
   - **Chore name** (e.g., "Walk dog")
   - **How often**: `daily`, `weekly`, or `asNeeded`
   - **Who**: `Ella`, `Hamish`, `either`, or `rotating`
   - **Day rule** (optional): `odd-Ella,even-Hamish`
   - **Description** (what the chore involves)

### Examples:

**Weekly Dog Walk (Rotates):**
```
5 | Walk dog | weekly | rotating | odd-Ella,even-Hamish | Monday dog walk
```

**Daily Homework Check (Specific Child):**
```
6 | Homework | daily | Ella | | Check and complete homework
```

**As-Needed Task (Either Child):**
```
7 | Vacuum living room | asNeeded | either | | Vacuum when needed
```

## üìä Weekly Review (Sunday Routine)

1. **Open Parent View** on the app
2. **Check the week totals**
   - Is it roughly equal?
   - Are there patterns? (e.g., Ella does more Mon-Wed, Hamish more Thu-Sat)
3. **Have a quick family chat**
   - Acknowledge effort
   - Adjust if needed
4. **Reset expectations** for the new week

## ‚ö†Ô∏è Common Issues

### "Chore button not working"
- Check internet connection
- Try refreshing the page
- If still broken, check Google Sheet is accessible

### "I did a chore but it's not showing"
- Check you tapped the right button
- Look in Google Sheet CompletedLog
- Parents can manually add if needed

### "Wrong person assigned to cat chores"
- Check today's date (odd vs. even)
- Rule is by day of month, not day of week

### "Dispute about who did more"
- Open Google Sheet CompletedLog
- Filter by date range
- Count actual entries (facts trump feelings!)

## üéì Teaching Accountability

### Week 1-2: Training Phase
- Check in daily with kids
- Remind them to log chores
- Celebrate when they remember

### Week 3-4: Independence
- Let kids manage their own logging
- Review weekly
- Only intervene if obvious gaps

### Week 5+: Autonomous
- Kids check app themselves
- Parents review weekly only
- System becomes habit

---

## üí° Tips for Success

1. **Add to Home Screen** - makes it feel like a real app
2. **Check weekly, not daily** - trust the system
3. **Use data in discussions** - "Let's look at the log" ends arguments
4. **Adjust chores as needed** - easy to add/remove in the sheet
5. **Celebrate effort** - acknowledge when totals are roughly equal
